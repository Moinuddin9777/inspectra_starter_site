# Product Context for Naming, Branding, and Symbol Discovery

## Product Vision

The current product is known internally as Web Component Inspector, but the long-term vision extends far beyond component inspection.

The platform is evolving into a unified Developer Experience and Runtime Intelligence Platform that helps developers observe, understand, debug, analyze, and optimize applications directly from their development environment.

The goal is to eliminate context switching between browsers, debuggers, network tools, coverage reports, accessibility scanners, terminals, and IDEs.

The platform brings all of these capabilities into a single intelligent workspace.
